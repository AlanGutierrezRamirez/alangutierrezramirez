My Projects

Shelley Manor

As a member of an 18-person student team at ESAT, I was a key contributor to Shelley Manor, an award-winning game now published on Steam. Developed using Unreal Engine 5, the project involved a collaborative effort of 8 programmers, 8 artists, and 3 game developers. My work on this project, along with the in-game sounds and music provided by Berklee students, led to the game winning The Best Student Game at the Latam Games Com 2025. This experience demonstrates my ability to thrive in a large, multidisciplinary team environment to deliver a polished, commercial-ready product.

JAM Engine

As part of a final project at ESAT, I developed a highly efficient 3D graphics engine from scratch using OpenGL. This engine was engineered with an Entity Component System (ECS) for a highly performant and flexible architecture. Key features include a fly camera for seamless navigation, dynamic lighting, and advanced volumetric lighting. This project demonstrates my deep understanding of core rendering principles and system-level design.
